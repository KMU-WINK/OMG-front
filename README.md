# OMG - front
2022-09 ~ 2023-01 WINK 팀 프로젝트 One More Glass (OMG)


## 개발주제
- 유엔 지속 가능한 발전 17가지 목표 (UN-SDGs) 중 2가지 이상 해당되는 주제
- 게시글 등록 및 수거로 다양한 병을 분리배출 하기 위한 환경관련 플랫폼 제작


## 개발환경
- VS Code, WebStorm 등
- React.js
- [OMG-back](https://github.com/KMU-WINK/OMG-back)
- MySQL
